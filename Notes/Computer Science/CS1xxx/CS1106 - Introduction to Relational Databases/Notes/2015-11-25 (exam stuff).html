Sample Paper

Question 2 was on table creation and manipulation, rather than select queries.

Question 3 was an SQL question, with six parts in increasing order of difficulty.

Question 1

Question 2

Example code:

CREATE TABLE weatherstats
(station VARCHAR(20),
day DATE,
max_temp FLOAT(3,1),
min_temp FLOAT(3,1),
rainfall FLOAT(4,1),
PRIMARY KEY (station, day)
);
